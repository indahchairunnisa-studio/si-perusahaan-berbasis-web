# About perusahaan-berbasis-web
A web-based **Company Information System** built using PHP Native without a framework. This system helps manage employees, departments, projects, and payroll with a simple and user-friendly interface.

## **ğŸš€ Key Features:**
  - **User Login:** Secure login system for authorized users.
  - **Employee Management:** Add, view, edit, and delete employee data.
  - **Department Management:** Organize and manage departments within the company.
  - **Project Management:** Track projects and assign employees.
  - **Payroll Management:** Manage employee salaries and payments.
  - **Dashboard Overview:** Summary of employees, projects, and departments in one place.
